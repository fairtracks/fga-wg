    Class: Bundle¶

    A bundle representing a set of genome annotation files, organised in sub-collections. Metadata has been harmonised in line with the "FAIRification of Genomic Annotations" data model.

    URI: https://w3id.org/fga-wg/schema/bundle/Bundle

    Slots¶

    Name Cardinality and Range Description Inheritance
    bundle_metadata 1
    BundleMetadata    		 Top-level metadata about the bundle of genomic annotation files. direct
    donors *
    Donor    		 Information about the donors or complete organisms from which the samples were taken. direct
    experiments *
    Experiment    		 Information about sequencing experiments that have been carried out to generate the files. direct
    files *
    File    		 Information about particular genome annotation (and other relevant) files. direct
    file_collections *
    FileCollection    		 Information about collections of files contained in this dataset, each collection defined according to some selection criteria. direct
    analyses *
    Analysis    		 Information about computational processing and analyses that have been carried out to generate the files. direct
    samples *
    Sample    		 Information about the biospecimens/samples used as raw material for lab experiments. direct
    studies *
    Study    		 The scientific studies, i.e. units of research, within which experiments and/or analyses have been carried out. direct
